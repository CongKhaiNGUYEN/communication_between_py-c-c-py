# python game online communication

## complie peer.c
`
gcc peer.c -o peer -pthread
`
## how to make communication
### on first computer
on first terminal
`
python3 py.py
`
<br />
open second terminal
`
./peer
`

### on second computer
on first terminal
`
python3 py.py
`
<br />
open second terminal
`
./peer
`

### finally
press `enter` in each python process
